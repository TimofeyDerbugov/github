Курс: Введение в GitHub

Что такое GitHub?

GitHub — это веб-платформа для хостинга и совместной работы над проектами с использованием системы контроля версий Git. Она позволяет разработчикам хранить код, отслеживать изменения и работать вместе над проектами.

Основные понятия

  • Git — распределённая система контроля версий.
  • Repository (репозиторий) — место, где хранится проект.
  • Commit (коммит) — сохранение изменений в истории проекта.
  • Branch (ветка) — параллельная линия разработки.
  • Pull Request — запрос на объединение изменений из одной ветки в другую.

Шаги для начала работы

1. Создание аккаунта

Зарегистрируйтесь на GitHub.

2. Создание репозитория

Нажмите кнопку "New" на главной странице, укажите имя репозитория, описание (по желанию), выберите публичный или приватный и нажмите "Create repository".

3. Клонирование репозитория

git clone https://github.com/ваш_профиль/имя_репозитория.git

4. Добавление и коммит изменений

git add .
git commit -m "Описание изменений"

5. Отправка изменений на GitHub

git push origin main

Работа с Pull Request

Внесите изменения в отдельной ветке, создайте Pull Request через интерфейс GitHub, обсудите и слейте изменения после проверки.

Итоги

GitHub — мощный инструмент для совместной разработки. Основные операции: clone, add, commit, push, pull request. Практика — лучший способ освоить работу с GitHub.
